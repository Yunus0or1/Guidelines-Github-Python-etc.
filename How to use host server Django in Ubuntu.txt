
### Django Install :
pip3 install Django==1.11.9
pip3 install mysql-python


### MySQL install first :: 

sudo apt-get install mysql-server
sudo apt-get install libmysqlclient-dev
sudo /etc/init.d/mysql stop

Stop Auto Start

sudo nano /etc/init/mysql.conf
Comment out the start on line near the top of the file, the start on might be spread across two lines, so comment out both. (comment adding # at the beginning)


Then :



### Xampp :

chmod 755 xampp.run

sudo ./xampp.run -- install

To start xampp :

sudo /opt/lampp/manager-linux-x64.run
sudo /opt/lampp/xampp stop

sudo /opt/lampp/uninstall  -- uninstall xampp



###  Nginx 

stop two ports 

sudo fuser -k 80/tcp
sudo fuser -k 443/tcp


sudo systemctl start nginx
sudo systemctl stop nginx
systemctl status nginx
